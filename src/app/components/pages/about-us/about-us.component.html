About Us

The main objective of this web is to provide the hand free access to the library portal through web interface.

We have a big collection of books with the options of retrieving the details of the books that available in the library

and borrow them.

How to borrow a book

In order to borrow a book you will need to be logged, if you dont have account you will need to register.

In the Books section you will find all the books available in the library. you will have the options to filter the books by

(Names, Genres, Year, Pages, Languages, Authors). each book has two buttons "Info" that will provide the information

and "Borrow" that will let you borrow the book. when you will find the book you want to borrow you will only need to click

on "Borrow".

How to return a book

In the Borrows section you will find all the borrows you have made.

you will have all the details of each borrow you made. when a borrow has one day till the return the borrow row will be red.

in order to return the book you will need to click on this button assignment_returned to return the book.
